# Tasty Bites – Basic Interactive Restaurant Menu

A responsive single-page restaurant menu application built with HTML, CSS, and JavaScript.
The project focuses on dynamic rendering, filtering functionality, and clean semantic structure.

## Live Demo

https://kaedin-restaurant-filter.netlify.app/

## Features

- Dynamic menu rendering using JavaScript
- Category-based filtering system
- Event delegation for efficient event handling
- Responsive grid layout using CSS Grid
- Semantic and accessible HTML structure
- Smooth UI animations for menu items

## Technologies Used

- HTML5
- CSS (Grid, Flexbox, Responsive Design)
- JavaScript (DOM manipulation, events)

## Concepts Practiced

- Rendering UI from data (array -> DOM)
- Event handling and delegation
- State management for active filters
- Usage of classes for creating and storing menu items
- Responsive design that avoids unnecessary media queries

## Challenges & What I Learned

- Managing UI state when filtering items
- Avoiding layout shifts and visual jank during rendering
- Understanding the difference between data-driven UI vs DOM manipulation
- Structuring code for readability and maintainability
